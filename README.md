# User Account Management System (React + JavaScript)
This project is a simple **user account management system** built with **React (JavaScript)**.  
It allows users to:
- Register an account
- Log in
- Manage and update their account details
- Log out  
- Persist data using `localStorage` (simulated database)
- 
### User Authentication
- Login using email + password
- Validate credentials from stored users
- Auto-login if session exists

### Registration
- Create a new user with name, email, password
- Prevent duplicate email registration
- Save users in `localStorage`

### Account Management
- View logged-in user details
- Update name & email
- Automatically sync updates to storage
- Persist updated user session

### Navigation
Includes a responsive **Navbar**:
- Login
- Register
- My Account (only when logged in)
- Logout
